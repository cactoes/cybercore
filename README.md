# cybercore
a project inspired by cybercore

## included
- themes
    - light
    - dark
- extension
    - session timer in the bottom right
    - line : column numbers
- user snippets
    - c++
    - typescript

## supported languages
these are supported others might not work properly
* c/c++
* java/type-script
* java
* markdown
* ini
* toml
* html/css
* cmake
* json
